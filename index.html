MPGram Web · лёгкая веб-версия на MadelineProto

Быстро. Экономно. Самостоятельный деплой.

Веб-оболочка Telegram на базе MadelineProto, оптимизированная для минимального потребления трафика и ресурсов сервера: кеширование, gzip, экономичный пуллинг и готовые Docker-конфиги.

Быстрый старт (Docker) Что внутри

Эта страница статична и может быть размещена на GitHub Pages (github.io). Само приложение требует сервер (PHP + nginx).

Кеш и статика

Долгий cache-control для JS/CSS/картинок, immutable; gzip включён.

Экономный пуллинг

Пауза при скрытой вкладке, бэкофф на ошибках, меньше запросов.

OPcache + JIT

Горячий кеш PHP-кода, отключённые таймстемпы для максимума хитов.

Особенности

  • Работает через MadelineProto
  • Поддержка чатов, каналов, вложений
  • Адаптивный интерфейс под мобильные браузеры

Требования

  • nginx + PHP-FPM (или Docker)
  • PHP 8.x, расширения mbstring, json
  • API ключи Telegram (MadelineProto)

Оптимизации

  • Кеширование статики 30 дней, immutable
  • Отключение access_log для снижения I/O
  • Пониженная активность JS при неактивной вкладке

Установка (Docker)

В каталоге docker/ выполните сборку и запуск контейнеров:

cd docker
docker compose up -d --build

# Приложение будет доступно на http://localhost/

Для HTTPS поместите сертификаты в docker/nginx/ssl и используйте конфиг https.conf.

Как использовать

  1. Разверните серверную часть (см. Docker-инструкции выше).
  2. Откройте страницу входа /login.php на своём домене.
  3. Авторизуйтесь и используйте лёгкий веб-клиент.

GitHub Pages размещает только этот статический лендинг. Сам MPGram Web (PHP) запускается на вашем сервере.
